# multi-html

> 传统页面, 多页面管理, 前端分离模式

## Environment

`Node >= 10.13.0`

## Start

 - 克隆此项目: 
```bash
git clone https://github.com/zwfec/multiHtml-webpack.git
```
 - 在项目目录下, 执行命令安装相关依赖:

``` bash
yarn install
```

## Develop

``` bash
# 运行开发环境
npm run dev
```

## Build

``` bash
# 构建并生成正式文件, 项目文件目录: dist
npm run build:ok
```
